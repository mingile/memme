# MemMe

> 기록을 기억으로 남기는 Chrome 사이드패널 메모장

## ✨ Overview

MemMe는 Chrome 사이드패널 기반의 미니멀 메모 확장 프로그램입니다.

브라우저에서 글을 읽다가,
유튜브를 보다가,
심지어 크롬만 켜놓고 멍때리는 와중에도,
문득 떠오르는 생각을 바로 기록할 수 있습니다.

## 🚀 Features

- 빠른 메모 작성
- 자동 저장
- 기억 리스트 관리
- 수정 / 삭제
- 라이브러리 모드

## 🛠 Tech Stack

- React
- TypeScript
- Plasmo
- Chrome Extension Manifest v3

## 📦 Installation (Development)

```bash
pnpm install
pnpm dev
```
